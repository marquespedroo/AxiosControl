#### D. Global Indices

**GSI (Global Severity Index) - Overall psychological distress:**
```
Total_sum = SUM(All 53 items)
Total_items_answered = COUNT_NON_ZERO(All 53 items) + COUNT(Additional items)
GSI = Total_sum / (53 + 4)  # 53 primary items + 4 additional items
GSI = Total_sum / 57
```

**PST (Positive Symptom Total) - Number of symptoms endorsed:**
```
PST = COUNT_IF(All 53 items, value > 0)
```

**PSDI (Positive Symptom Distress Index) - Intensity of symptoms:**
```
PSDI = Total_sum / PST
```

#### E. T-Score Conversion

All dimension means, GSI, PST, and PSDI are converted to T-scores using normative tables:

# VLOOKUP function (approximate match)
def vlookup(lookup_value, table, return_column_index, approximate=True):
    """
    Simulates Excel's VLOOKUP with approximate match (sorted ascending)
    table: list of tuples [(key1, val1_col1, val1_col2, ...), ...]
    """
    if not approximate:
        # Exact match
        for row in table:
            if row[0] == lookup_value:
                return row[return_column_index - 1]
        return None
    else:
        # Approximate match - find largest value <= lookup_value
        result = None
        for row in table:
            if row[0] <= lookup_value:
                result = row[return_column_index - 1]
            else:
                break
        return result
